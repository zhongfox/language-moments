# 3.10 Hash

哈希是一种数据结构, 它维护了一个键值对集合, 而且将每个键都与一个值关联起来. 哈希也被称为映射(map), 有时也叫作关联数组(associative array)

|          | Ruby | Javascript                      | Lua | Java | Go |
|----------|------|---------------------------------|-----|------|----|
| 实现     | Hash | Object                          |     |      |    |
| 字面量   |      | `{"age": 8, name: "fox"}`       |     |      |    |
| 属性读取 |      | `person["age"]`<br>`person.age` |     |      |    |
