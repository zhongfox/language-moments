# 4.1 Naming conventions

|          | Ruby            | Javascript | Lua             | Java      | Go        |
|----------|-----------------|------------|-----------------|-----------|-----------|
| 命名约定 | underscore_case | camelCase  | underscore_case | camelCase | camelCase |

虽然各语言没有强制要求以上命名约定, 但是用户程序命令, 和语言内置函数/标准库的函数和变量命名格式保持一致, 是一种不错的最佳实践
